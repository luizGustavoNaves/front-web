Principais formatações de texto:

Negrito / Destaque:

Nessa frase, temos um termo em negrito usando a tag B (não semântica).

Nessa frase, temos um termo em destaque usando a tag STRONG (que é semântica).

Itálico / Ênfase:

Nessa frase, temos um termo em Itálico usando a tag I (não semântica).

Nessa frase, temos um termo em ÊNFASE usando a tag EM (semântica).

Texto marcado:

Podemos criar também um texto MARCADO usando a tag MARK.

Outra maneira de modificar o style (cor).

Texto grande e texto pequeno:

Estamos criando um texto grande e um texto pequeno.

Texto deletado:

Podemos maracr um texto como excluído para indicar que ele deve ser lido mas não considerado.

Texto inserido:

Podemos maracr um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois (semântico).

Existe também o sublinhado com a tag U (não semântico).

Texto sObrescrito:

Para inserir coisas do tipo x20+3.

Texto sUbscrito:

Para inserir coisas do tipo H20.
